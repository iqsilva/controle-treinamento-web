Sobre nós

Somos uma corporação que nasceu de um projeto de faculdade para poder suprir as necessidades de uma empresa de linhas aéreas em realizar o controle de treinamento de seus funcionários, criando assim um software que gerencie esses treinamentos e facilitando a gestão de cada um dos funcionários.

Atualmente temos 3 versões de softwares, sendo as seguintes: - Azul 1.0 em C - Azul 2.0 em JAVA - Azul 3.0 em WEB Cada uma pode ser personalizada de acordo com as necessidades do cliente.
